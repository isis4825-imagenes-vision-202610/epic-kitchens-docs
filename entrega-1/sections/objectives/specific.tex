\section{Specific Objectives}

\begin{enumerate}

\item To analyze the structure of the \href{https://epic-kitchens.github.io/}{\textit{EPIC-KITCHENS-100}} dataset and the relevance matrices used in the multimodal retrieval challenge, identifying the organization of video segments, textual descriptions, and their semantic relationships.

\item To design a deep learning architecture that integrates temporal visual representations and semantic language representations to project both modalities into a shared feature space.

\item To implement and train the proposed model to estimate semantic similarity between video--text pairs and generate retrieval rankings for both search directions.

\item To evaluate the model's performance using ranking metrics such as \textit{Normalized Discounted Cumulative Gain} (nDCG) and \textit{Mean Average Precision} (mAP), comparing the results against baseline approaches reported in the literature.

\end{enumerate}
